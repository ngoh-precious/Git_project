Installing Kubernetes

We'll take a look at some of the ways to install and get started with kubernetes
We'll look at:

  • Play with kubernetes
  • Using Minikube to install kubernetes on a laptop
  • Installing kubernetes in the Google Cloud with the Google Container Engine (GKE)
  • Installing kubernetes on AWS Using the kops tools
  • Installing kubernetes manually using kubeadm

Firstly, the are a lot of ways to install kubernetes. The options I've chosen for this are the once I think will be the most useful.

Secondly, kubernetes is a fast-growing project. This means some of what we'll discuss here will change.
